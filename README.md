## Featured Project

### LabFlow

LabFlow is a deployed full-stack laboratory project management application for university research labs. It helps teams manage projects, tasks, experiments, protocols, equipment bookings, review workflows, project membership, role-based access control, and experiment-linked notebook entries.

**Tech stack:** React, Vite, Ant Design, Node.js, Express, PostgreSQL, Sequelize, JWT authentication, Jest, Supertest

**Key features:**

Role-based access control for admin, supervisor, and researcher users
Project membership and permission-aware workflows
Task completion review
Experiment and protocol review workflows
Equipment booking conflict prevention
Experiment-linked notebook entries
Sequelize migrations
Backend security hardening
36 passing automated backend tests across 7 test suites
Deployment with Vercel, Render, and Neon PostgreSQL

Live demo: https://labflow-brown.vercel.app
Repository: https://github.com/prndavis77/labflow
