## Featured Project

### LabFlow

LabFlow is a full-stack laboratory project management application for university research labs. It helps teams manage projects, tasks, experiments, protocols, equipment bookings, review workflows, and experiment-linked notebook entries.

**Tech stack:** React, Vite, Ant Design, Node.js, Express, PostgreSQL, Sequelize, JWT authentication

**Key features:**

* Role-based access control
* Project membership and permission-aware workflows
* Task completion review
* Experiment and protocol review workflows
* Equipment booking conflict prevention
* Experiment-linked notebook entries
* Deployment with Vercel, Render, and Neon PostgreSQL

Live demo: https://labflow-brown.vercel.app
Repository: https://github.com/prndavis77/labflow
